Palindrome checker

Create a .net core C# website that checks if a user input string is a palindrome.

  • Your solution will include a web front end created with bootstrap 3 or 4 and your preferred js library,
  • The backend will be a .net core web api project using your favourite dependency injection tool and ORM.
  • The solution should persist to local db, all successfully entered palindromes and provide a screen that lists all known palindromes from the persisted list.
  • Bootstrap for layout and styling

Some sample palindromes

  • Was it a cat I saw?
  • Don't nod.
  • Radar
  • No lemon, no melon
